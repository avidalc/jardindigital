# Jardín Digital

Proyecto: Jardín Digital

Descripción:
Un proyecto para gestionar y documentar plantas, jardinería y notas de observación — "Jardín Digital". Este README provee instrucciones básicas para instalar, ejecutar y contribuir.

Estado:
- Inicial / en desarrollo

Requisitos:
- Node.js >= 16 (si aplica)
- Python 3.8+ (si aplica)
- Docker (opcional)

Instalación (ejemplo genérico):
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/avidalc/jardindigital.git
   cd jardindigital
   ```
2. Instalar dependencias (ajustar según el stack):
   - Node.js/npm:
     ```bash
     npm install
     ```
   - Python/venv:
     ```bash
     python -m venv venv
     source venv/bin/activate
     pip install -r requirements.txt
     ```
3. Configurar variables de entorno (crear `.env` a partir de `.env.example` si existe).

Ejecución (ejemplo genérico):
```bash
# Con Node
npm start
# O con Python
python run.py
```

Contribuir:
1. Fork del repositorio
2. Crear una rama con tu feature: `git checkout -b feature/nombre`
3. Hacer commit de tus cambios
4. Abrir un pull request
